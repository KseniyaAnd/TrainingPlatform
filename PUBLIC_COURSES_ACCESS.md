# Публичный доступ к странице "Все курсы"

## Выполненные изменения

### 1. Header компонент (`src/app/layout/header/header.ts`)

**До:**

```typescript
navigateToCourses(): void {
  void this.router.navigate([this.isAuthenticated() ? '/courses' : '/login']);
}
```

**После:**

```typescript
navigateToCourses(): void {
  void this.router.navigate(['/courses']);
}
```

**Результат:** Кнопка "Все курсы" теперь всегда ведет на `/courses`, независимо от статуса авторизации.

### 2. Courses страница (`src/app/pages/courses/courses.ts`)

#### Добавлен флаг авторизации:

```typescript
readonly isAuthenticated = this.authState.isAuthenticated;
```

#### Обновлена логика загрузки курсов:

**Для неавторизованных пользователей:**

- ✅ Показываются все доступные курсы
- ❌ Не показываются кнопки "Записаться"/"Отписаться"

**Для авторизованных студентов:**

- ✅ Показываются все курсы с информацией о записи
- ✅ Доступны кнопки "Записаться"/"Отписаться"
- ✅ Доступен scope "me" (мои курсы)

**Для авторизованных преподавателей:**

- ✅ Показываются все курсы
- ✅ Доступен scope "me" (созданные курсы)

#### Код изменений:

```typescript
private async loadPage(cursor: string | null): Promise<void> {
  // ...
  const isAuthenticated = this.isAuthenticated();

  // Проверка для "Мои курсы"
  if (scope === 'me' && !isAuthenticated) {
    this.error.set('Необходимо войти в систему для просмотра своих курсов');
    return;
  }

  // Загрузка всех курсов - доступно для всех
  if (!scope || scope !== 'me') {
    if (isAuthenticated && role === 'STUDENT' && !cursor) {
      // Для студентов - с информацией о записи
      const coursesWithEnrollment = await firstValueFrom(
        this.coursesService.loadCoursesWithEnrollmentStatus({ limit: 20, cursor, q }),
      );
      // ...
    } else {
      // Для неавторизованных и преподавателей - просто список курсов
      const response = await firstValueFrom(
        this.coursesService.getCourses({ limit: 20, cursor, q }),
      );
      // ...
    }
  }
}
```

## Маршруты

| Маршрут                   | Доступ                 | Описание        |
| ------------------------- | ---------------------- | --------------- |
| `/courses`                | 🌐 Публичный           | Все курсы       |
| `/courses?scope=me`       | 🔒 Требует авторизации | Мои курсы       |
| `/courses?q=search`       | 🌐 Публичный           | Поиск курсов    |
| `/courses?tag=javascript` | 🌐 Публичный           | Фильтр по тегу  |
| `/courses/:id`            | 🌐 Публичный\*         | Детали курса    |
| `/courses/:id/student`    | 🔒 Только студенты     | Материалы курса |

\*Детали курса могут требовать авторизацию в зависимости от настроек курса
