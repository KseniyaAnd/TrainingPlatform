# Реализация Footer компонента

## Описание

Создан простой, минималистичный футер с прозрачным фоном для Learning Platform.

## ✅ Успешно реализовано

### Дизайн

- **Прозрачный фон** - `bg-transparent`
- **Тонкая верхняя граница** - `border-t border-gray-200`
- **Адаптивная верстка** - мобильная и десктопная версии
- **Минималистичный стиль** - простой и чистый дизайн

### Структура

- **Copyright** - автоматический год через `new Date().getFullYear()`
- **Навигация** - ссылки на основные разделы:
  - Курсы
  - О платформе
  - Контакты
  - Конфиденциальность

### Технические детали

- Standalone компонент
- `ChangeDetectionStrategy.OnPush` для оптимизации
- Tailwind CSS для стилизации
- RouterLink для навигации
- Flexbox для адаптивной раскладки

## Интеграция

Футер добавлен в `app.html` с использованием flexbox layout:

```html
<div class="flex min-h-screen flex-col">
  <app-header />
  <main class="flex-1">
    <router-outlet />
  </main>
  <app-footer />
</div>
```

### Преимущества такой структуры:

- `min-h-screen` - минимальная высота экрана
- `flex-col` - вертикальная раскладка
- `flex-1` на main - контент занимает всё доступное пространство
- `mt-auto` на footer - футер прижимается к низу

## Адаптивность

### Мобильные устройства (< 640px)

- Вертикальная раскладка
- Центрированный текст
- Ссылки в одну строку с переносом

### Десктоп (≥ 640px)

- Горизонтальная раскладка
- Copyright слева, навигация справа
- Текст выровнен по краям

## Стилизация

### Цвета

- Текст: `text-gray-600`
- Граница: `border-gray-200`
- Hover: `hover:text-gray-900`

### Отступы

- Внутренние: `py-6` (вертикальные)
- Контейнер: `px-4` (горизонтальные)
- Между элементами: `gap-4` / `gap-6`

### Эффекты

- Плавный переход цвета при наведении: `transition-colors`

## Использование

Футер автоматически отображается на всех страницах приложения.

Для изменения ссылок отредактируйте `footer.component.html`:

```html
<a routerLink="/your-page" class="transition-colors hover:text-gray-900"> Ваша ссылка </a>
```
