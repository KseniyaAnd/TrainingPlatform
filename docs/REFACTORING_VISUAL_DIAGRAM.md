```
course-slider.css
┌─────────────────────────────────────────────────────────────────┐

│    background-color: #34d399 !important;                        │
│    color: #ffffff !important;                                   │
│  }                                                               │
│                                                                  │
│  Проблемы:                                                       │
│  ⚠️ ::ng-deep deprecated                                        │
│  ⚠️ !important - плохая практика                                │
└─────────────────────────────────────────────────────────────────┘
```

### ✅ Предлагаемая структура

**Вариант 1: ViewEncapsulation.None**

```typescript
@Component({
  selector: 'app-course-slider',
  encapsulation: ViewEncapsulation.None,
  styleUrls: ['./course-slider.css'],
})
export class CourseSliderComponent {}
```

```css
/* course-slider.css */
.app-course-slider .glider-track {
  gap: 1.5rem;
}

.app-course-slider .glider-dot {
  @apply w-2.5 h-2.5 rounded-full bg-gray-300 transition-all;
}

.app-course-slider .glider-dot.active {
  @apply w-6 rounded-md bg-teal-500;
}
```

**Вариант 2: Глобальные стили**

```css
/* styles.css */
.glider-track {
  @apply gap-6;
}

.glider-dot {
  @apply w-2.5 h-2.5 rounded-full bg-gray-300 transition-all cursor-pointer;
}

.glider-dot:hover {
  @apply bg-gray-400;
}

.glider-dot.active {
  @apply w-6 rounded-md bg-teal-500;
}

/* PrimeNG customization */
.p-tag {
  @apply bg-brand-mint text-white border-0;
}
```

**Преимущества:**

- ✅ Нет deprecated API
- ✅ Правильная инкапсуляция
- ✅ Использование Tailwind @apply
- ✅ Нет !important
