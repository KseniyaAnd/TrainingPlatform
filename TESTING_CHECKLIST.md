# Чеклист тестирования унифицированного компонента курсов

## Подготовка

- [ ] Запустить приложение: `npm start` или `ng serve`
- [ ] Открыть браузер на `http://localhost:4200`
- [ ] Подготовить два аккаунта: студент и преподаватель

---

## Тесты для СТУДЕНТА

### Навигация - Все курсы

- [ ] Перейти на главную страницу
- [ ] Нажать "Курсы" или перейти на `/courses`
- [ ] ✅ Отображается список всех доступных курсов
- [ ] ✅ Для каждого курса видна кнопка "Записаться" или "Отписаться"
- [ ] ✅ Кнопка "Create course" НЕ отображается

### Навигация - Мои курсы

- [ ] Нажать "Мои курсы" в меню
- [ ] ✅ URL изменился на `/courses?scope=me`
- [ ] ✅ Отображаются только курсы, на которые студент записан
- [ ] ✅ Кнопки "Записаться"/"Отписаться" НЕ отображаются
- [ ] ✅ Кнопка "Create course" НЕ отображается

### Запись на курс

- [ ] Перейти на `/courses` (все курсы)
- [ ] Найти курс, на который НЕ записан
- [ ] Нажать "Записаться"
- [ ] ✅ Кнопка изменилась на "Отписаться"
- [ ] ✅ Курс появился в "Мои курсы" (`/courses?scope=me`)

### Отписка от курса

- [ ] Перейти на `/courses` (все курсы)
- [ ] Найти курс, на который записан
- [ ] Нажать "Отписаться"
- [ ] ✅ Кнопка изменилась на "Записаться"
- [ ] ✅ Курс исчез из "Мои курсы" (`/courses?scope=me`)

### Обратная совместимость

- [ ] Вручную перейти на `/my-courses/learning`
- [ ] ✅ Автоматически перенаправляет на `/courses?scope=me`
- [ ] ✅ Отображаются мои курсы

## Тесты фильтрации (для обеих ролей)

### Фильтр по тегу

- [ ] Перейти на `/courses?tag=фронтенд`
- [ ] ✅ Отображаются только курсы с тегом "фронтенд"
- [ ] Изменить тег в URL на другой
- [ ] ✅ Список курсов обновляется без перезагрузки страницы

### Поиск

- [ ] Перейти на `/courses?q=Angular`
- [ ] ✅ Отображаются только курсы с "Angular" в названии
- [ ] Изменить поисковый запрос в URL
- [ ] ✅ Список курсов обновляется без перезагрузки страницы

### Комбинация фильтров

- [ ] Перейти на `/courses?scope=me&q=Angular`
- [ ] ✅ Отображаются только мои курсы с "Angular" в названии

- [ ] ✅ Список дополнительно фильтруется по тегу
