# Исправление: Одинаковая ширина карточек в гриде

## Проблема

Карточки курсов имели разную ширину в гриде из-за `min-width: 350px` в CSS компонента `course-card`.

### Визуальная проблема:

```
До:
┌─────────────┐  ┌──────┐
│   Card 1    │  │Card 2│
│  (350px+)   │  │(auto)│
└─────────────┘  └──────┘
```

## Причина

В файле `src/app/components/course-card/course-card.css`:

```css
.course-card {
  display: flex;
  flex-direction: column;
  height: 120px;
  min-width: 350px; /* ← Проблема! */
  overflow: hidden;
}
```

`min-width: 350px` заставлял карточки иметь минимальную ширину 350px, что конфликтовало с CSS Grid и приводило к неравномерному распределению ширины.

## Решение

### 1. Обновлен CSS карточки

**Файл:** `src/app/components/course-card/course-card.css`

**До:**

```css
:host {
  display: block;
  height: 100%;
}

.course-card {
  display: flex;
  flex-direction: column;
  height: 120px;
  min-width: 350px;
  overflow: hidden;
}
```

**После:**

```css
:host {
  display: block;
  height: 100%;
  width: 100%; /* ← Добавлено */
}

.course-card {
  display: flex;
  flex-direction: column;
  height: 120px;
  width: 100%; /* ← Изменено с min-width: 350px */
  overflow: hidden;
}
```

### 2. Явно указан grid-cols-1 для мобильных

**Файл:** `src/app/pages/courses/courses.html`

**До:**

```html
<div class="grid gap-6 sm:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4"></div>
```

**После:**

```html
<div class="grid gap-6 grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4"></div>
```

## Результат

Теперь все карточки имеют одинаковую ширину в гриде:

```
После:
┌──────────┬──────────┬──────────┬──────────┐
│  Card 1  │  Card 2  │  Card 3  │  Card 4  │
│ (равная) │ (равная) │ (равная) │ (равная) │
└──────────┴──────────┴──────────┴──────────┘
```

### Как работает CSS Grid:

```css
/* Grid контейнер */
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 равные колонки */
  gap: 1.5rem;
}

/* Карточка занимает всю ширину своей ячейки */
.course-card {
  width: 100%; /* Растягивается на всю ширину ячейки */
}
```

## Breakpoints

### Mobile (< 640px)

```
grid-cols-1
┌─────────────────┐
│    Card 1       │
│   (100% width)  │
└─────────────────┘
```

### Tablet (640px - 1023px)

```
sm:grid-cols-2
┌─────────┬─────────┐
│ Card 1  │ Card 2  │
│ (50%)   │ (50%)   │
└─────────┴─────────┘
```

### Desktop (1024px - 1279px)

```
lg:grid-cols-3
┌──────┬──────┬──────┐
│Card 1│Card 2│Card 3│
│(33%) │(33%) │(33%) │
└──────┴──────┴──────┘
```

### Large Desktop (≥ 1280px)

```
xl:grid-cols-4
┌─────┬─────┬─────┬─────┐
│Card1│Card2│Card3│Card4│
│(25%)│(25%)│(25%)│(25%)│
└─────┴─────┴─────┴─────┘
```

## Преимущества

✅ **Одинаковая ширина** - все карточки равномерно распределены  
✅ **Адаптивность** - работает на всех размерах экрана  
✅ **Предсказуемость** - CSS Grid автоматически распределяет пространство  
✅ **Чистый код** - нет хаков или фиксированных размеров

## Тестирование

### Проверьте на разных размерах:

- [ ] Mobile (375px) - карточки занимают 100% ширины
- [ ] Tablet (768px) - 2 карточки равной ширины
- [ ] Desktop (1280px) - 3 карточки равной ширины
- [ ] Large Desktop (1920px) - 4 карточки равной ширины

### Проверьте с разным контентом:

- [ ] Короткие названия курсов
- [ ] Длинные названия курсов
- [ ] Много тегов
- [ ] Мало тегов
- [ ] С кнопками записи
- [ ] Без кнопок

Все карточки должны иметь одинаковую ширину независимо от контента!

## Дополнительные улучшения

Если нужно установить минимальную ширину для карточек на очень больших экранах:

```css
/* Опционально: ограничить максимальную ширину контейнера */
.courses-container {
  max-width: 1920px; /* Максимум 4 карточки по ~450px */
  margin: 0 auto;
}
```

Или использовать `auto-fit` для автоматического количества колонок:

```html
<!-- Альтернативный подход -->
<div class="grid gap-6" style="grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));"></div>
```

Но текущее решение с явными breakpoints более предсказуемо и контролируемо.
