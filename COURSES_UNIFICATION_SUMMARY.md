## Как это работает

### Единый компонент для всех ролей

Компонент `CoursesPage` теперь обрабатывает все сценарии:

1. **Все курсы**: `/courses`
2. **Мои курсы**: `/courses?scope=me`
   - Для студентов: показывает курсы, на которые записан
   - Для преподавателей: показывает созданные курсы + кнопка "Create course"
3. **Фильтрация**: `/courses?tag=фронтенд`
4. **Поиск**: `/courses?q=Angular`
5. **Комбинация**: `/courses?scope=me&q=Angular`

### Обратная совместимость

Старые URL автоматически перенаправляются:

- `/my-courses/teaching` → `/courses?scope=me`
- `/my-courses/learning` → `/courses?scope=me`
- `/my-courses/teaching/create` → `/courses/create`

## Примеры использования

### В шаблонах

```html
<!-- Все курсы -->
<a routerLink="/courses">Все курсы</a>

<!-- Мои курсы -->
<a [routerLink]="['/courses']" [queryParams]="{ scope: 'me' }">Мои курсы</a>

<!-- Создать курс -->
<a routerLink="/courses/create">Создать курс</a>

<!-- С фильтром -->
<a [routerLink]="['/courses']" [queryParams]="{ tag: 'фронтенд' }">Фронтенд</a>
```

### В компонентах

```typescript
// Все курсы
this.router.navigate(['/courses']);

// Мои курсы
this.router.navigate(['/courses'], { queryParams: { scope: 'me' } });

// Создать курс
this.router.navigate(['/courses/create']);

// С фильтрами
this.router.navigate(['/courses'], {
  queryParams: { scope: 'me', q: 'Angular' },
});
```
