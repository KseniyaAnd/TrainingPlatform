# Поиск на всю ширину Header

## Изменение

Поисковая строка теперь занимает всю доступную ширину в Header, обеспечивая лучший UX.

---

## До изменения

```
┌─────────────────────────────────────────────────────────────────┐
│ Logo  [Все курсы]  [Поиск...]  [Войти] [Зарегистрироваться]   │
└─────────────────────────────────────────────────────────────────┘
     ↑       ↑           ↑              ↑            ↑
   fixed   fixed    фиксированная    fixed        fixed
                       ширина
```

**Проблема:** Поиск имел фиксированную ширину и не использовал доступное пространство

---

## После изменения

```
┌─────────────────────────────────────────────────────────────────┐
│ Logo  [Все курсы]  [========== Поиск... ==========]  [Войти]   │
└─────────────────────────────────────────────────────────────────┘
     ↑       ↑                    ↑                        ↑
   fixed   fixed          растягивается                 fixed
                         на всю ширину!
```

**Решение:** Поиск занимает всё доступное пространство между кнопками

---

## Код изменений

### 1. SearchBar Component (`search-bar.html`)

**До:**

```html
<div class="relative flex-1 min-w-0 px-2">
  <!-- поиск -->
</div>
```

**После:**

```html
<div class="relative w-full">
  <!-- поиск -->
</div>
```

**Изменения:**

- ❌ Убрали `flex-1` (управление шириной теперь в родителе)
- ❌ Убрали `px-2` (отступы теперь в родителе)
- ✅ Добавили `w-full` (100% ширины родителя)

---

### 2. Header Component (`header.html`)

**До:**

```html
<div class="flex items-center gap-3 h-20 px-6">
  <a>Logo</a>
  <p-button>Все курсы</p-button>
  <app-search-bar />
  <app-user-menu />
</div>
```

**После:**

```html
<div class="flex items-center gap-3 h-20 px-6">
  <a>Logo</a>
  <p-button styleClass="shrink-0">Все курсы</p-button>

  <!-- Обертка для поиска с flex-1 -->
  <div class="flex-1 min-w-0">
    <app-search-bar />
  </div>

  <app-user-menu />
</div>
```

**Изменения:**

- ✅ Добавили `shrink-0` к кнопке "Все курсы" (не сжимается)
- ✅ Обернули `<app-search-bar />` в `div` с `flex-1 min-w-0`
- ✅ Поиск теперь занимает всё доступное пространство

---

## CSS классы

### `flex-1`

```css
flex: 1 1 0%;
```

Элемент растягивается, занимая всё доступное пространство

### `min-w-0`

```css
min-width: 0;
```

Позволяет элементу сжиматься меньше своего контента (важно для flex)

### `shrink-0`

```css
flex-shrink: 0;
```

Элемент не сжимается при нехватке места

### `w-full`

```css
width: 100%;
```

Элемент занимает 100% ширины родителя

## Технические детали

### Flexbox Layout

```html
<div class="flex">
  <!-- Flex контейнер -->
  <div class="shrink-0">
    <!-- Не сжимается -->
    Logo
  </div>
  <div class="shrink-0">
    <!-- Не сжимается -->
    Кнопка
  </div>
  <div class="flex-1">
    <!-- Растягивается! -->
    <div class="w-full">
      <!-- 100% родителя -->
      Поиск
    </div>
  </div>
  <div class="shrink-0">
    <!-- Не сжимается -->
    Меню
  </div>
</div>
```

### Расчет ширины

```
Ширина Header = 1200px (пример)
- Logo: 150px (shrink-0)
- Кнопка: 100px (shrink-0)
- Меню: 200px (shrink-0)
- Gap: 3 × 12px = 36px
= Поиск: 1200 - 150 - 100 - 200 - 36 = 714px ✅
```
