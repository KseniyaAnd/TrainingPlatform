# Быстрый старт: Админ-панель

## Доступ к админ-панели

1. Войдите в систему с учетной записью, имеющей роль `ADMIN`
2. В правом верхнем углу в меню пользователя появится кнопка **"Админ панель"**
3. Нажмите на кнопку для перехода на дашборд

## Основные функции

### 📊 Дашборд

**URL:** `/admin`

Отображает ключевую статистику:

- Количество пользователей
- Количество курсов
- Количество записей на курсы
- Средний балл по заданиям

Кликните на карточки для быстрого перехода к управлению.

### 👥 Управление пользователями

**URL:** `/admin/users`

**Возможности:**

- Просмотр всех пользователей
- Фильтрация по ролям (Все / Администраторы / Преподаватели / Студенты)
- Просмотр детальной информации о пользователе

**Как использовать:**

1. Выберите фильтр по роли (по умолчанию "Все")
2. Нажмите "Детали" для просмотра подробной информации

### 📋 Детали пользователя

**URL:** `/admin/users/:userId`

**Отображается:**

- Основная информация (имя, email, роль, дата регистрации)
- Записи на курсы с прогрессом
- Ответы на задания с оценками
- Созданные курсы (для преподавателей)

### 📚 Управление курсами

**URL:** `/admin/courses`

**Возможности:**

- Просмотр всех курсов платформы
- Редактирование любого курса
- Удаление курсов

**Действия:**

- **Просмотр** - открыть страницу курса
- **Редактировать** - открыть курс в режиме редактирования
- **Удалить** - удалить курс (с подтверждением)

⚠️ **Внимание:** Удаление курса необратимо и удалит все связанные данные!

## Навигация

- Используйте кнопку **"Назад"** для возврата на предыдущую страницу
- Кнопка **"Админ панель"** в меню всегда доступна для быстрого возврата на дашборд

## Безопасность

- Все страницы админ-панели защищены и доступны только пользователям с ролью `ADMIN`
- При попытке доступа без прав происходит автоматический редирект на главную страницу
- Все действия выполняются от имени авторизованного администратора

## API эндпоинты

Админ-панель использует следующие API эндпоинты:

### Эксклюзивный для админа:

```
GET /platform-statistics
```

### Пользователи:

```
GET /users?role={ADMIN|TEACHER|STUDENT}
GET /users/{userId}
GET /users/{userId}/details
```

### Курсы:

```
GET /courses
GET /courses/{courseId}
PATCH /courses/{courseId}
DELETE /courses/{courseId}
```

### AI аналитика:

```
GET /courses/{courseId}/ai/student-analytics
GET /courses/{courseId}/students/{studentId}/ai-study-plan
```

## Советы

1. **Фильтрация пользователей** - используйте фильтры для быстрого поиска нужной группы пользователей
2. **Детали пользователя** - проверяйте активность студентов через раздел "Ответы на задания"
3. **Управление курсами** - перед удалением курса убедитесь, что это действительно необходимо
4. **Статистика** - регулярно проверяйте дашборд для мониторинга активности платформы
